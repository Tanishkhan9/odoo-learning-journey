📚 Student Management Module

A comprehensive educational module for managing student information, enrollments, and academic records within Odoo. This module is designed as a practical learning resource for Odoo development.

✨ Features

📋 Student Records

Comprehensive student profiles with personal information, contact details, and academic status tracking.

📖 Course Enrollment

Easy enrollment management allowing students to be assigned to multiple courses with tracking capabilities.

🎓 Status Management

Track student lifecycle through different statuses: New, Active, Inactive, Graduated, and Dropped Out.

🔒 Security & Access Control

Role-based access with Student User and Student Manager groups, ensuring data security and proper authorization.

📊 Multiple Views

Multiple view types including Form, List, Search, and Kanban views for different use cases.

🚀 Key Components

Models:
  • student.student - Main student model with all business logic
Views:
  • Form View - Detailed student information with tabs
  • List View - Student directory with filtering
  • Search View - Advanced search and filtering options
  • Kanban View - Card-based student directory
Features:
  • Automatic age calculation from birth date
  • Enrollment tracking with course count
  • Status workflow (New → Active → Graduated)
  • Email and registration number validation
  • Audit trail with creation and modification timestamps

👥 User Roles

Student Manager - Full CRUD permissions on student records
Student User - Read-only access to student records

🎯 Use Cases

  • Educational institution student management
  • Training center enrollment tracking
  • Corporate learning management
  • Demonstration of Odoo development practices

📝 Sample Data

The module comes with sample student records for testing and demonstration purposes. These records include:

  • 8 pre-created student profiles
  • Various status examples (Active, New, Graduated, Inactive)
  • Different enrollment scenarios

🔧 Installation

  1. Place module in your Odoo addons directory
  2. Update App List in Odoo Settings
  3. Search for "Student Management"
  4. Click Install
  5. Sample data will be automatically loaded

💡 Learning Resources

This module serves as a comprehensive learning resource covering:

  • Model definition and inheritance
  • ORM operations (CRUD)
  • View design and customization
  • Security and access control
  • Computed fields and constraints
  • Business logic implementation
